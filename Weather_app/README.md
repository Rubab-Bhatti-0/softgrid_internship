# Weather App

A simple weather application that displays current weather information for a selected city.

## Features

- Search weather by city name
- Display temperature, humidity, wind speed, and weather conditions
- Responsive user interface

## Installation

1. Clone the repository.
2. Install dependencies (if applicable):
   - For JavaScript/Node projects, run `npm install`.
3. Run the application:
   - For web projects, open `index.html` in a browser or run a local server.

## Usage

1. Enter a city name in the search field.
2. Submit the form to view current weather details.

## Notes

- Ensure any required API keys are configured in the application code.
